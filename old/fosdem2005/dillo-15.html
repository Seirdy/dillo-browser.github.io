Current features (most important)

  • A very good subset of HTML 4.01 SPEC is supported
  • Image formats support: PNG, JPEG and GIF
  • Streamed rendering, including tables
  • An extraordinary small size (around 40 times less than big browsers)
  • Remarkably fast rendering (near 100KB/sec on a 233Mhz Pentium I)
  • Snappy response even on a 75Mhz/32MB Pentium
  • Extensible with plugins (plugins can use dillo as an API)
  • It has an ordered, clean and documented code base that's easy to extend and maintain
