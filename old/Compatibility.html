Compatibility

Dillo has been reported to run on the following operating systems / architectures. If you have Dillo running on something not listed here, please tell us!

Dillo3

Dillo3 uses FLTK-1.3.

GNU/Linux:

  • x86
  • amd64
  • arm: Cortex-A8
  • mips64el
  • IRIX 6.5.27f

*BSD

  • DragonFlyBSD
  • OpenBSD
  • NetBSD, NetBSD/VAX
  • FreeBSD

OS X:

  • 10.6.7
  • 10.5.8 ppc32

Miscellaneous:

  • Cygwin (mostly working)

Dillo2

Dillo2 used FLTK2.

GNU/Linux:

  • x86
  • x86-64
  • ARM XScale Linksys NSLU2, ARM926EJ-S i.MX27
  • m68k
  • PowerPC Gentoo, iMac G5 with Crux-PPC
  • MIPS 4KEc (using nxlib!)
  • Blackfin BF537 (µClinux, nxlib)
  • Nios II DE2-70 (µClinux, nxlib)
  • Pandora info

*BSD

  • DragonFlyBSD
  • OpenBSD
  • FreeBSD
  • NetBSD on Sun Ultra 1, iMac macppc

OS X:

  • Mac OS X (but FLTK2 was buggy on OSX)

Solaris:

  • Solaris

AIX:

  • IBM RS6000/42T with PowerPC 604 CPU (AIX 5.1ML9)

Handhelds:

  • iPAQ [some instructions] [screenshot]
  • Openmoko Neo FreeRunner [package] [screenshot]

Dillo1

Dillo1 (0.8.x) used GTK+ 1.2.

GNU/Linux:

  • Intel x86 CPU-family, GNU/Linux (libc5 and glibc2)
  • DEC Alpha (glibc 2.0)
  • PPC (YellowDogLinux 2.1; Mandrake Linux 9.1; Gentoo 1.4)
  • Sparc 5 (32 bits), glibc 2.1.3, gcc 2.95.3
  • Sparc ultra 1 (64 bits kernel, 32 bits userspace), glibc 2.1.3, gcc 2.95.3
  • StrongARM (33MB RAM, armedslack/gcc-3.4)

*BSD

  • NetBSD (in pkgsrc)
  • FreeBSD (in the ports)
  • OpenBSD (in the ports)
  • DragonFlyBSD (out of the box).

OS X:

  • MAC OS X
  • GNU Darwin (Power PC and x86)

Solaris:

  • Sun Blade 100 (Ultra SPARC III), Solaris 8, gcc 3.0
  • SunOS jaca 5.7 Generic_106541-04 sun4u sparc
  • Sparc ultra 10 (64 bits) solaris 8, gcc 2.95.2
  • Solaris 2.6 (SunOS 5.6), gcc 3.3, Sparc & UltraSparc
  • Sun Ultra 5, Solaris 10, 64bit sun4u, gcc 3.4.3

Handhelds:

  • iPAQ handheld! (familiar distribution)
  • Psion handheld 5MX/5MXPro (CPU: 32bit ARM 710T, 37MHz) --Debian ARM linux
  • Intel's StrongArm CPUs!
  • GMATE's Yopy
  • NetBSD, which includes dillo, has been ported to several StrongARM and MIPS based handhelds.
  • Simputer (encore; picopeta)
  • iRex iLiad

Miscellaneous:

  • SGI O2 running IRIX
  • Tru64 (OSF1) 4.0 on Alpha
  • HP 9000 715/80 with Debian hppa
  • 68k Linux on Q40 and Q60 (sinclair QL compatible machines)
  • QNX RTP 6.1/x86 (with occasional memory faults out of GTK+)
  • AIX 4.3 (with some tweaks)
  • SONY PlayStation2 (Linux)
  • Atari-based 68k-systems running MiNT
  • Microsoft Windows using Cygwin
  • RISC OS!
